# DATA DESCRIPTION

## 1. District-Level Efficiency Panel

**Files:** `data/raw/table3_dea_bootstrap_full.csv`, `data/raw/table4_sfa_district_te.csv`

**Source:** Official district-level agricultural statistics compiled from the State Statistical Committee of the Republic of Azerbaijan, covering 67 districts over the period 2000-2024 (N = 1,523 district-year observations in the full production panel).

**Key variables:**

| Variable | Description |
|---|---|
| `region` | District name |
| `year` | Calendar year |
| `vrs_raw` | Raw (uncorrected) DEA technical efficiency score, variable returns to scale |
| `vrs_bc` | Bootstrap-corrected DEA technical efficiency score (Simar & Wilson smoothed bootstrap, 200 replications) |
| `sfa_te_translog` | Translog stochastic frontier technical efficiency score |

These files are regenerated by re-running the bootstrap DEA and SFA estimation procedures described in Section 2.2 of the manuscript against the official panel. Re-running these procedures reproduces the previously reported summary statistics closely (raw DEA mean 0.591; bootstrap-corrected DEA mean 0.420; SFA mean TE 0.797; SFA log-likelihood -453.23), as verified in `output/Table2_replication_efficiency.csv` after running `02_table2_replication_efficiency.py`.

## 2. District-Adjacency Classification

**Location:** hard-coded in `code/_paths.py` (`TREATED`, `ADJACENT_1ST_ORDER`, `ADJACENT_2ND_ORDER`).

**Source:** Official administrative border descriptions verified against each district's dedicated record, citing Azerbaijan's State Statistical Committee and the relevant presidential decrees (7 July 2021 territorial-administrative reform; 5 December 2023 decree adding Aghdara). An automated GIS approach (GADM v4.1) was attempted first and rejected as unreliable for this application; see `GADM_ISSUES.md` for full details.

**Treated districts (9, matching the companion event-study treatment group):** Aghdam, Fuzuli, Gubadli, Jabrayil, Kalbajar, Khojavand, Lachin, Shusha, Zangilan.

**Reintegrated territory used to define adjacency (10, includes Aghdara):** the nine treated districts plus Aghdara, which is part of the reintegrated territory but excluded from the formal treatment group because only a single post-reintegration observation is available for it.

**First-order adjacent districts (7):** Aghjabadi, Barda, Beylagan, Dashkasan, Goranboy, Goygol, Tartar. (Khojaly officially borders several reintegrated districts but is excluded because the production panel has no usable observations for it.)

**Second-order adjacent districts (8):** Zardab, Imishli, Agdash, Yevlakh, Samukh, Shamkir, Gadabay, Ganja city.

Per-district sourcing detail is documented in Table 1 (reproduced by `01_table1_adjacency_classification.py`).

## 3. Post-Treatment Period

`Post = 1` for years 2021 onward (the first full year following the November 2020 Nagorno-Karabakh ceasefire), consistent with the companion event-study specification.
